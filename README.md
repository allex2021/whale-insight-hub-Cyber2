# 🐋 Whale Intelligence Pro

> Real-time crypto whale tracking terminal with AI confluence signals — built by a self-taught developer from Bangladesh.

🔴 **Live Demo:** [whale-insight-hub.lovable.app](https://whale-insight-hub.lovable.app)

---

## What is this?

A free, open-source trading intelligence terminal that gives retail traders 
access to the same whale tracking data that institutional traders pay 
$999/month for (Nansen, Glassnode).

Built with zero funding, zero university degree, from Khulna, Bangladesh.

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** TanStack Start + Supabase
- **Data:** Binance WebSocket, Hyperliquid API, Deribit, CoinGecko
- **AI:** Gemini via Lovable Gateway
- **Hosting:** Lovable.dev

---

## Why free?

Most whale tracking tools cost $150–$999/month and target institutional 
investors. This project exists to democratize access to these signals 
for retail traders, especially in emerging markets.
